推荐方式（macOS 12 请用 pip，勿用 brew yt-dlp）：

pip install -U yt-dlp
yt-dlp --version

安装脚本已自动 pip 安装；路径可填 .venv/bin/yt-dlp 的完整路径。

brew install yt-dlp
yt-dlp --version

或使用项目脚本：./scripts/install-mac.sh

pip install -U yt-dlp

# 或 Scoop
scoop install yt-dlp

# 或 Chocolatey
choco install yt-dlp

安装后可在命令行执行 yt-dlp --version 验证。
