推荐方式，与 MyDownloader 同一 Python 环境：

pip install -U yt-dlp
yt-dlp --version

配置中路径填写 yt-dlp 即可（若在 PATH 中）。

brew install yt-dlp
yt-dlp --version

或使用项目脚本：./scripts/install-mac.sh

pip install -U yt-dlp

# 或 Scoop
scoop install yt-dlp

# 或 Chocolatey
choco install yt-dlp

安装后可在命令行执行 yt-dlp --version 验证。
