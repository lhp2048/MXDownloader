项目已提供 compose，生产环境推荐：

cd family-mediacenter
docker compose up -d aria2

默认 RPC：http://127.0.0.1:6800/jsonrpc，Secret：family-mediacenter（与 compose 中 RPC_SECRET 一致）。

brew install aria2

# 本机 RPC（需自行保持运行）
aria2c --enable-rpc --rpc-listen-port=6800 \
  --rpc-listen-all=false --continue

或：./scripts/install-mac.sh --with-docker

scoop install aria2
# 或
choco install aria2

# 启动 RPC（示例）
aria2c --enable-rpc --rpc-listen-port=6800

确保 RPC 端口与上方配置一致，Secret 两边配置相同。
