Alist 建议通过 Docker 部署：

cd family-mediacenter
docker compose up -d alist

# 同时启动 aria2 + alist
docker compose up -d
  1. 访问 http://127.0.0.1:5244
  2. 首次启动查看容器日志获取管理员密码
  3. 登录后台 → 设置 → 获取 Token
  4. 在本页填写服务地址与 Token，并勾选「启用」
./scripts/install-mac.sh --with-docker

或单独：docker compose up -d alist。详见项目 docs/DEPLOY_MAC.md。
